# 章节格式参考

下面是常见的中文长篇小说章节命名约定，便于 `/目录` 工具在生成 章节目录.md 时选择合适的形式。

## 单层编号（适合 20-30 章中小型小说）

```markdown
## 章节目录

- 第 1 章：开篇
- 第 2 章：触发
...
- 第 20 章：尾声
```

适用于：校园、都市、言情、玄幻短篇。

## 双层编号（适合 30-80 章中长篇）

```markdown
## 章节目录

- 第 1 章：开篇
- 第 2 章：触发
...

## 第二卷：对抗

- 第 21 章：中点
- 第 22 章：高潮
```

适用于：玄幻升级、历史穿越、悬疑长篇。

## 三层卷章（适合 80+ 章超长篇）

```markdown
## 第一卷：建置

- 第 1 章：开篇
- 第 2 章：触发

## 第二卷：对抗

- 第 21 章：中点
...

## 第五卷：尾声

- 第 121 章：新世界
```

适用于：超长篇网络连载、史诗级玄幻/科幻。

## 章节标题命名约定

- 中文为主，长度 4-8 字
- 避免使用纯数字或英文
- 关键情节点可用四字成语
- 章末留 1-2 章做"钩子"（给下一章留悬念）

## 反例

- "第 1 章"（无标题）
- "Chapter 1: Beginning"（混合中英文）
- "第 1 章 第 1 节 子标题"（三层嵌套）
- 超长标题（一句话作标题）
