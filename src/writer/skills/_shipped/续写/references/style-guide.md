# 续写文风指南

下面是 `/续写` 工具在生成续写段落时的文风、语气、世界观一致性提示词模板。

## 文风提示词模板

```
你的写作风格必须严格匹配项目已有章节：

- 叙述视角：第三人称限定视角（主角内心独白 + 客观外部描写）
- 时态：过去时
- 句子长度：中等（15-30 字为主，避免长复合句）
- 对话风格：符合角色身份（武将说话短促、文人说话引经据典、女性角色细腻）
- 比喻偏好：避免现代网络用语，使用古典意象
```

## 角色一致性检查清单

- 主角说话方式是否与已有章节一致？
- 反派是否还在坚持"坏但有逻辑"的人设？
- 配角是否保持了他们的"标签特征"（如爱喝酒、爱唠叨等）？
- 主角的能力成长曲线是否合理（不会突然会原本不会的招数）？

## 世界观一致性

- 朝代/时代背景不要混用（唐朝不能出现明朝官职）
- 修炼等级体系不要前后矛盾（如果体系是"炼气-筑基-金丹"，不要突然冒出"元婴"）
- 地理距离符合设定（不要"千里之外"然后"半日可达"）

## 续写段落长度

| 场景 | 长度 | 说明 |
| --- | --- | --- |
| 日常推进 | 500-800 字 | 不需要大事件 |
| 情节转折 | 800-1200 字 | 有明确的事件或冲突 |
| 高潮段落 | 1200-2000 字 | 关键对决、真相揭露 |
| 过渡段 | 300-500 字 | 场景切换、时间跳跃 |

## 续写后状态标记

在草稿末尾插入以下 marker 便于后续识别：

```markdown
<!-- CONTINUATION-POINT: 2026-07-09T15:30:00 -->
```

下次续写时检测这个 marker，从 marker 位置之后开始写。

## 章节完成标记

当章节写完时：

```markdown
<!-- CHAPTER-COMPLETE: 2026-07-09T16:00:00 -->
```

并在 `chapter_summaries.json` 里更新摘要 + `completed: true`。
