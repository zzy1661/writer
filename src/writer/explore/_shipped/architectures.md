# 精选写作架构

本目录精选八种适合长篇小说项目的整体架构，供 `/init` explore 模式推荐。

## 雪花写作法

由一句话核心逐层扩写，像雪花生长一样把人物、冲突、情节与细节铺成完整全书，适合长篇、多伏笔和复杂剧情。

### 完整六层实操步骤

1. **一句话核心故事**：用一句话讲清谁经历了什么冲突，最终得到或失去了什么。
2. **四段式短梗概**：把核心拆成开端、发展、高潮、结局，锁定故事脉络。
3. **人物小传搭建**：完善主角、配角、反派的性格、执念、弱点、目标和人物弧光。
4. **章节长梗概**：将四段脉络扩写成全书各大篇章，标注转折、低谷、爽点和伏笔。
5. **细化场景与支线**：穿插支线剧情、人物互动、伏笔铺垫与冲突细节。
6. **完整成文**：根据细化大纲填充对话、动作、环境和心理描写。

### 适配场景与优缺点

适配所有题材，尤其适合长篇小说、系列故事、剧情复杂和多伏笔反转的作品。优势是逻辑严谨、伏笔容易前后呼应；缺点是前期大纲耗时较长，不适合即兴短篇。

## 三幕结构

把故事切为铺垫、对抗、解决三幕，结构简单、节奏稳定，是小说、网文和影视剧本都通用的基础架构。

1. **第一幕·铺垫（25%）**：出场人物、日常世界、触发事件和抉择关口。
2. **第二幕·对抗（50%）**：主角进入新世界，遭遇层层障碍、中点反转与至暗时刻。
3. **第三幕·解决（25%）**：最终决战、真相揭晓，以及新旧生活的对比收尾。

### 衍生变体：四幕结构

可以把三幕中间的上升与下降拆开，多安排一次大转折，适合节奏复杂、多反转的悬疑小说。

## 英雄之旅

以神话原型的成长闭环组织冒险故事，天然适配玄幻、奇幻、武侠、史诗和成长流网文。

### 十二阶段

平凡世界 → 召唤冒险 → 拒绝召唤 → 遇见导师 → 跨越门槛 → 试炼之路 → 接近洞穴 → 终极考验 → 获取奖励 → 归途重生 → 携赐回归。

### 适配与限制

优点是自带成长爽感，读者代入强；都市日常和纯情感故事使用时需要避免过度套路化。

## 三步八段式

专为连载长篇设计，把故事推进拆成八个可执行段落，适合爽文、都市和玄幻题材。

1. **开端段**：铺垫背景，交代主角困境。
2. **转机段**：金手指或关键机遇出现。
3. **成长段**：第一次小崛起，解决小矛盾。
4. **冲突段**：强敌登场，危机爆发。
5. **低谷段**：主角惨败，失去重要事物。
6. **蜕变段**：闭关、顿悟或策略转变，完成升级。
7. **决战段**：正面对抗核心反派。
8. **收尾段**：复仇、登顶或完成阶段性目标，开启新篇章。

## 三明治写作法

先分层控制主线剧情、人物支线与单章细节，再把三层夹写在一起，适合多角色群像长篇。

### 三层结构

1. **顶层**：主线剧情大纲，确定全书方向。
2. **中层**：人物弧光、支线和伏笔，填充故事的夹心。
3. **底层**：单章细节、场景和对话，把计划落地成可写内容。

### 写作流程

先搭主线，再穿插人物支线，最后填充章节细节。它和雪花写作法互补，兼顾剧情控制与人物塑造。

## 人物弧光架构

不以事件为唯一主轴，而以主角的内心变化搭建整体框架，适合现实向、言情、治愈和文学小说。

### 人物变化路径

初始性格 → 外部冲击 → 内心挣扎 → 认知崩塌 → 重塑自我 → 最终人格。

### 适配场景

故事中的外部事件应持续迫使人物做出选择，让情节服务于人物的改变，最终以新的价值判断或人格状态收束。

## 起承转合

用“起、承、转、合”四步快速搭建故事，门槛低、推进清晰，适合短篇、新手入门、散文和极简故事。

1. **起**：引入人物背景和初始处境。
2. **承**：让矛盾逐步发展，积累压力。
3. **转**：安排重大转折或危机，改变故事方向。
4. **合**：让冲突尘埃落定，并完成收尾或升华。

## 单元串联架构

以一条大主线贯穿全书，中间由相对独立的单元小故事串联，每个单元都有自己的起承转合，适合连载长篇和轻小说。

### 组织方式

大主线负责持续的终极目标与世界变化；每个单元负责一个局部任务、人物关系或阶段性谜题，并在结尾留下通往主线的钩子。

### 适配场景

适合无限流、闯关、校园日常，以及医生、律师等职业网文。单元边界清晰，便于稳定连载，同时保留长期追读的主线悬念。
