# 大纲输出示例

下面是两条已生成大纲的真实样例，用于参考 SKILL.md 输出格式。

## 示例 1：穿越历史题材

输入：`/大纲 一个现代程序员穿越到唐朝贞观年间`

```
# 长安程序员

## 前提

现代程序员李承乾意外穿越到唐朝贞观年间，凭借对历史的了解和编程思维，在长安城中立足。

## 四幕大纲

- 第一章：猝死程序员的贞观之夜（建置）
- 第二章：初入长安（建置）
- 第三章：识破玄武门旧事（建置）
- 第四章：长孙无忌的邀约（触发事件）
- 第五章：东宫夜谈（第一情节点）
- 第六章：入仕为官（对抗）
- 第七章：朝堂博弈（对抗）
- 第八章：突厥南下（对抗）
- 第九章：兰州大捷（中点）
- 第十章：吐蕃和亲（中点）
- 第十一章：长孙之死（转折）
- 第十二章：太子之乱（转折）
- 第十三章：真相揭露（第二情节点）
- 第十四章：玄武门之变（高潮）
- 第十五章：新唐盛世（解决）
- 第十六章：尾声·程序员回归（尾声）
```

## 示例 2：玄幻升级流

输入：`/大纲 一个被退婚的废柴少年意外得到上古剑灵传承`

```
# 剑逆苍穹

## 前提

被未婚妻当众退婚的废柴少年叶尘，意外在家族禁地中得到上古剑灵传承，从此踏上修炼之路。

## 四幕大纲

- 第一章：当众退婚（建置）
- 第二章：禁地奇遇（建置）
- 第三章：剑灵觉醒（第一情节点）
- 第四章：入门测试（对抗）
- 第五章：外门弟子（对抗）
- 第六章：核心弟子之战（中点）
- 第七章：遗迹探险（中点）
- 第八章：真相浮现（转折）
- 第九章：身世之谜（转折）
- 第十章：家族存亡（第二情节点）
- 第十一章：决战剑冢（高潮）
- 第十二章：飞升上界（解决）
```

## 命名约定

- 中文书名（《长安程序员》《剑逆苍穹》）
- 章节标题用四字短语为主
- 每个情节点有明确章节
- 解决幕可加尾声
