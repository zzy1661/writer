# 四幕大纲模板

四幕结构是经典的故事节奏模板，适用于大多数长篇小说。每个项目可根据题材调整章节数。

## 第一幕：建置（Setup）

占总篇幅约 20-25%。作用：

- 介绍主角和日常世界
- 建立主角的"缺失"（缺乏 / 渴望 / 问题）
- 引出触发事件（inciting incident），打破主角的日常平衡

示例节奏：

- 第 1-3 章：日常世界 + 主角人设
- 第 4-5 章：触发事件，主角面临新挑战
- 第 5 章末尾：第一情节点（Plot Point 1），主角决定踏上旅程

## 第二幕：对抗（Rising Action）

占总篇幅约 35-40%。作用：

- 主角尝试解决问题，遇到层层阻力
- 引入盟友与反派，丰富人物关系
- 通过中点（midpoint）翻转局势

示例节奏：

- 第 6-15 章：主角应对挑战
- 第 15 章：中点，主角看似要成功（或失败）
- 第 15-20 章：阻力升级，反派展现真正实力

## 第三幕：转折（Crisis）

占总篇幅约 20-25%。作用：

- 主角最低谷（all is lost moment）
- 关键信息揭露或角色觉醒
- 第二情节点（Plot Point 2），主角做出最终决定

示例节奏：

- 第 20-22 章：主角彻底失败，丧失信心
- 第 22 章：核心真相揭露
- 第 23 章：主角觉醒，决战准备

## 第四幕：解决（Resolution）

占总篇幅约 15-20%。作用：

- 高潮对决
- 主线收束
- 新平衡建立（"日常世界 2.0"）

示例节奏：

- 第 24-25 章：高潮 + 决战
- 第 25 章末尾：主线收束
- 第 26 章：尾声（可选）+ 新日常建立
