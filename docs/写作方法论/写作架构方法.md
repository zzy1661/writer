# 主流小说/故事整体架构写作法（除雪花写作法）

分四大类：**人物驱动架构法、情节结构法、大纲分层法、极简实操法**，覆盖网文、长篇小说、短篇、剧本通用。

## 一、经典三幕结构（最通用，影视 \+ 小说通用）

核心：把故事切为 \\_\\_ Setup（铺垫）→ Confrontation（对抗）→ Resolution（解决）\\_\\_

1. 第一幕・铺垫（25%）：出场人物、日常世界、触发事件（打破平淡）、抉择关口；

2. 第二幕・对抗（50%）：主角进入新世界，遭遇层层障碍、中点反转、低谷至暗时刻；

3. 第三幕・结局（25%）：最终决战、真相揭晓、新旧生活对比收尾。
   适配：长篇、短篇、剧本、网文主线框架。

### 衍生变体：四幕结构

在三幕中间拆分「上升 / 下降」，多一次大转折，适合节奏复杂、多反转悬疑小说。

## 二、英雄之旅（坎贝尔神话原型，史诗 / 玄幻 / 仙侠首选）

12 阶段闭环，所有冒险类故事底层逻辑：
平凡世界→召唤冒险→拒绝召唤→遇见导师→跨越门槛→试炼之路→接近洞穴→终极考验→获取奖励→归途重生→携赐回归
优点：天然适配玄幻、奇幻、武侠、成长流网文；
缺点：都市日常、纯情感文容易套路化。

## 三、三步八段式（网文专用高效架构）

专为连载长篇设计，适配爽文、都市、玄幻：

1. 开端段：铺垫背景，主角困境

2. 转机段：金手指 / 机遇出现

3. 成长段：第一次小崛起，解决小矛盾

4. 冲突段：强敌登场，危机爆发

5. 低谷段：主角惨败，失去一切

6. 蜕变段：闭关 / 顿悟，实力升级

7. 决战段：正面对抗核心反派

8. 收尾段：复仇 / 登顶，开启新篇章

## 四、布兰登・桑德森三明治写作法（同雪花作者，人物 \+ 情节双控）

和雪花法同源，但逻辑相反：雪花是**由大到小细化**，三明治是**分层夹写**。
三层结构：

1. 顶层：主线剧情大纲（面包上层）

2. 中层：人物弧光、支线、伏笔（夹心馅料）

3. 底层：单章细节、场景对话（面包下层）
   写作流程：先搭主线→穿插人物支线→填充章节细节，适合多角色群像长篇。

## 五、节拍表（布莱克剧本节拍表，精准控节奏）

共 15 个固定节拍，精确分配全书每一段功能，适合悬疑、言情、电影改编小说：
开篇画面、主题陈述、铺垫、触发事件、辩论、中点、虚假胜利 / 失败、至暗时刻、灵魂抉择、终局决战、结局画面等。
优势：不会节奏拖沓，精准控制爽点、虐点、反转位置。

## 六、七大情节架构（针对不同故事类型）

### 1\. 目标阻碍法（任务流）

核心公式：**主角明确目标→层层阻碍→克服阻碍→达成 / 失去目标**
适配：探险、破案、职场、夺宝文。

### 2\. 人物弧光架构（情感 / 文艺向）

不以事件为主，以**主角内心变化**搭建整体框架：
初始性格→外部冲击→内心挣扎→认知崩塌→重塑自我→最终人格
适配：现实向、言情、治愈、文学小说。

### 3\. 问题解决架构（推理 / 都市）

开篇抛出核心谜题 / 生存危机，全文围绕「寻找答案、化解危机」推进，所有情节服务解谜。

### 4\. 双线并行架构

主线 \+ 副线两条时间 / 人物线穿插，中期交汇，结尾收束；
常见：悬疑（过去真相线 \+ 现在查案线）、家庭文（两代人故事）。

### 5\. 环形叙事架构（文艺、悬疑反转）

首尾场景呼应，结局揭示开篇伏笔，时间线打乱重组；如《追忆似水年华》《盗梦空间》。

### 6\. 单元串联架构（网文长篇、轻小说）

大主线贯穿，中间由独立单元小故事串联，每个单元有独立小起承转合；
适配：无限流、闯关、校园日常、职业文（医生 / 律师）。

### 7\. 对立冲突架构

全程围绕两大阵营、两种理念、正反主角对抗搭建，冲突逐层升级。

## 七、极简轻量化架构（适合短篇、新手快速动笔）

### 1\. 起承转合（中式传统四步法）

起：引入人物背景
承：矛盾逐步发展
转：重大转折 / 危机
合：尘埃落定，收尾升华
古文、短篇、散文、小小说通用，门槛最低。

### 2\. 两点一线法

只抓两个核心点：**初始困境、最终结局**，中间一条冲突线填充，适合即兴写作。

## 八、反套路架构（非传统叙事）

1. 无主角架构：群像叙事，无单一核心人物（如《红楼梦》）

2. 反英雄结构：主角从好人逐步堕落，弧光向下

3. 开放式框架：不写完整结局，侧重过程与思考（文学向）

## 各方法对比速选指南

| 写作方法     | 适合题材           | 核心优势                     |
| ------------ | ------------------ | ---------------------------- |
| 三幕结构     | 全类型通用         | 简单好上手，节奏稳定         |
| 英雄之旅     | 玄幻 / 武侠 / 冒险 | 自带成长爽感，读者代入强     |
| 三明治架构   | 多角色长篇         | 和雪花法互补，兼顾剧情与人设 |
| 布莱克节拍表 | 影视改编、悬疑言情 | 精准卡点，无拖沓             |
| 单元串联     | 无限流、职业网文   | 连载友好，容易持续产出       |
| 人物弧光架构 | 现实、言情、文学   | 情感细腻，人物立体           |
| 起承转合     | 短篇、新手入门     | 极简零门槛                   |
