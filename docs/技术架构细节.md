### 一、设计概要(三大议题决定表)

| 议题 | 决策 | 替代 |
| --- | --- | --- |
| Q3 上下文拼装 | 静态骨架 + 动态 RAG top-up(混合)| 纯静态 / 纯动态 / 角色自适应子图 |
| Q3 滑动窗口 | 全量但按需金字塔(近 3 章 + 中 5-10 章 + 远 10+ 章实体卡)| 仅近 N 章 / 全量展开 |

| Q4 注册机制 | LangGraph `ToolNode` + `@tool` 装饰 | 自建 Registry / 混合 MCP |
| Q4 权限模型 | cwd 内免 confirm / cwd 外 confirm | 角色白名单 / 全 confirm |
| Q5 质控层数 | 3 道门 + 跨章节审计 | 单门 / 写中+写后 / 卷后自动 |
| Q5 自评维度 | 5 维度评分表(大纲/人物/伏笔/节奏/风格)| 单维度 / 三轮独立角色 |
